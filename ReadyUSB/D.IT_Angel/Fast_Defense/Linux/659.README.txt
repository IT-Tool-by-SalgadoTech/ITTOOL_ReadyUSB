 IT-Tool by SalgadoTech  -  FAST DEFENSE  -  Linux LAUNCHERS (IT_Script .txt)
================================================================================

45 IT_Script launchers (one per Fast Defense Linux script). Each .txt follows the
IT-Tool launcher format (#NOTE / WTIME / SCRIPT / ENTER) and, when executed by the
IT-Tool device on the target terminal, performs three things on any of the three
supported distros (Debian, Fedora, Arch):

  1) INSTALLS the packages that script needs, per distro
     (apt-get / dnf|yum / pacman), only if the tool is missing.
  2) DOWNLOADS the script from GitHub into  ~/ITTOOL  in the user's home
     (SUDO_USER-aware), together with the shared library _itlib.sh which every
     .sh sources - without it the scripts would not run.
  3) RUNS it  (bash for .sh, python3 for the .py monitors).

Scripts whose action targets a daemon (xrdp, firewalld, NetworkManager, ClamAV,
hostapd, snapper) do NOT auto-install that daemon: the .sh detects its presence
and degrades cleanly, so the launcher never disrupts a running system.

--------------------------------------------------------------------------------
 TESTED
--------------------------------------------------------------------------------
End-to-end on Debian 12, Fedora 44 and Arch (rolling): every launcher installs
its deps, downloads the script + _itlib.sh to ~/ITTOOL and runs it. Verified real
installs (ping, rfkill on Debian; psutil via dnf and pacman) and that _itlib.sh
lands next to each script so the scripts start correctly.
